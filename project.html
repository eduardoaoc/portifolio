Jogo de Luta

Um básico jogo de luta para ser jogado com dois jogadores em um mesmo teclado. Cada personagem tem dois poderes de ataque, além de skins características. Foi criado em forma de aprendizado para que eu entendesse um pouco mais sobre o pacote do Python chamado Pygame.

Formulário Lexus

Esse projeto é um formulário no qual, o cliente que foi atendido, dá o feedback sobre como foi o atendimento, seleciona o atendente, avalia o atendimento, etc. Caso há campos que não foram preenchidos, será apresentado na tela uma mensagem pedindo para que os campos sejam preenchidos. Feito para aprender um pouco como é a relação do front-end com o back-end.
